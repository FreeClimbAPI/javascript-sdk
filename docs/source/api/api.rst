setPersyUrl(newUrl)
^^^^^^^^^^^^^^^^^^^^^

    Change the Persephony instance the SDK interacts with. https://persephony.com/apiserver by default

    :newUrl: {string} The URL to the api of the Persephony instance to use. e.g https://persephony.com/apiserver
